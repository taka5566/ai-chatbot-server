# AI Chatbot Server

A versatile AI chatbot application with multiple personality modes, powered by Poe API and deployable on Vercel.

## Features

- 🤖 **Multiple AI Personalities**: Choose from Claude, Buddy, Assistant, or Spark
- 💬 **Conversation Memory**: Maintains context throughout the chat
- 🎨 **Modern UI**: Clean, responsive design with smooth animations
- ⚡ **Serverless Architecture**: Fast, scalable deployment on Vercel
- 🔒 **Secure**: API keys stored as environment variables

## AI Personalities

| Personality | AI Model | Trait |
|------------|----------|-------|
| **Claude** | Claude-Sonnet-4.6 | Helpful and thoughtful |
| **Buddy** | GPT-5.4 | Warm and encouraging |
| **Assistant** | Claude-Opus-4.7 | Formal and precise |
| **Spark** | Gemini-3.1-Pro | Imaginative and playful |

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js (Vercel Serverless Functions)
- **API**: Poe API (OpenAI-compatible endpoint)
- **Deployment**: Vercel

## Prerequisites

- Node.js 18+ installed
- Poe API key ([Get one here](https://poe.com/api_key))
- Vercel account (for deployment)
